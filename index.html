🛒 Flipkart Product Review Sentiment Analyzer

🔍 About This Project

This is a machine learning-based web app built using Python, Streamlit, and scikit-learn.

It analyzes product reviews from Flipkart and classifies them as Positive or Negative.

💡 Features

  • TF-IDF vectorization of text reviews
  • Naive Bayes and Random Forest classifiers
  • Clean and simple user interface
  • WordClouds, sentiment graphs, and confusion matrices
